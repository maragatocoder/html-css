Inserindo vídeos internos com HTML5

Os formatos de vídeos mais usados são MP4, M4V, WEBM e OGV.
Sempre inserir todos os formatos para rodar no navegador de todos os usúarios.
Para inserir vídeos internos vamos usar a tag <video>
Dentro da tag vídeo podemos definir o tamanho com a tag widht="..."
Podemos colocar uma capa para o vídeo usando a tag poster="..."
e também os controles de vídeo inserindo 'control' 'autoplay' e 'loop'.
